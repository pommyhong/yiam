# Yiam — Gamification / Habit Design

เป้า: ให้รู้สึก "ได้รางวัล" + อยากกลับมาทุกวัน → สร้างนิสัยการจด
ข้อจำกัด: ห้ามทรยศ **minimal + ไม่ตัดสิน (no-judgment)**

---

## หลักคิด
**Reward currency = "ความเข้าใจตัวเอง + ความต่อเนื่อง" ไม่ใช่แต้ม**
- gamify ที่ *feedback* (ความคืบหน้า, insight) ไม่ใช่สร้าง currency ปลอม
- intrinsic > extrinsic — รางวัลจริงคือ "ฉันคุมเงินตัวเองได้"

---

## เอา / ไม่เอา
```
เอา ✅                           ไม่เอา ❌
────────────────────────────────────────────────────
🔥 Streak + grace (อภัย 1 วัน)    🪙 Points / coins / XP
📅 Daily ritual "ปิดยอดวันนี้"     🏆 Leaderboard / social compare
⭕ Progress ring (จด X/30 วัน)     🎉 Confetti / มาสคอต / ป้ายแวววาว
🏅 Milestone เงียบ (การ์ดสะสม)     😟 ลงโทษตอน streak หลุด (= guilt)
💡 Variable reward (insight ใหม่)
🎁 Weekly recap (story)
```

## The daily loop (Hooked)
```
Trigger  เตือนสุภาพ "จดของวันนี้รึยัง" (ไม่จิก, ปิดได้)
Action   จด < 5 วิ
Reward   streak ติ๊ก + บางครั้งปลด insight/milestone (variable = เซอร์ไพรส์)
Invest   ยิ่งจด → insight แม่นขึ้น → เหตุผลกลับมา
```

## Mechanics → ฟีเจอร์ Yiam
```
1  Streak + grace      ชิป "🔥 7" บน Home · พลาดได้ 1 วัน (streak-freeze)
                       reason: streak ปกติทำคนเลิกตอนหลุด — grace = no-judgment version
2  Daily ritual        "ปิดยอดวันนี้" ปุ่ม/การ์ด · แม้วันนี้ใช้ ฿0 ก็ติ๊กได้
                       reason: Peak-End ทุกวัน + ทำให้ "ไม่มีรายจ่าย" ก็ยังมีส่วนร่วม
3  Progress ring       "เดือนนี้จด 18/30 วัน" วงแหวน mono บน Home/Stats
4  Milestone           7 วัน · 30 วัน · 100 รายการ · จดครบเดือนแรก
                       → การ์ดสะสมเงียบๆ (หน้า achievements) ไม่ใช่ป๊อปอัปแวววาว
5  Variable reward     insight ใหม่ปลดเมื่อข้อมูลพอ ("ปลดล็อกอินไซต์ใหม่")
6  Weekly recap        ทุกอาทิตย์ = ของขวัญ story-style (เด่นสุด 1 ตัวเลข/สัปดาห์)
```

## Design rule: ฉลองโดยไม่หลุด mono
- ฉลองด้วย **typography + พื้นที่ + 1 จังหวะ** ไม่ใช่สี
- reward moment = การ์ด/หน้า **ดำเต็ม** + ตัวเลขใหญ่มาก + 1 emoji (ใช้ระบบ mono ให้เด่น)
- motion 1 ครั้ง (number scale-in) ไม่ใช่ confetti
- ห้าม guilt: streak หลุด → "เริ่มใหม่ได้เสมอ" ไม่ใช่ "คุณพลาด!"

## Roadmap (gamification)
```
ทำแล้ว/รอบนี้   Reward moment (streak milestone, ดำเต็มจอ)
ถัดไป          ชิป streak บน Home · progress ring · daily ritual "ปิดยอด"
อนาคต          หน้า Achievements · Weekly recap story · insight-unlock
```
